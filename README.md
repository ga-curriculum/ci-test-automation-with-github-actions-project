## About

Build a tested FastAPI application and configure a continuous integration pipeline using GitHub Actions to automatically run your test suite with each push or pull request.

Estimated completion time is around **15 hours**.

### Prerequisites

- Creating and structuring a FastAPI project
- Writing unit and integration tests using pytest (and optionally httpx)
- Organizing test files and test suites
- Understanding the purpose of CI and automated test pipelines
- Reading and writing simple GitHub Actions YAML configuration files
- Using Git and GitHub to push code and trigger workflows
